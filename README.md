# 暴刷 LeetCode

这是一个面向 LeetCode 的系统化学习与刷题记录工程，按「编程语言 → 数据结构 → 算法思想」组织内容，目标是兼顾知识体系与实战题感。

## 学习路径

- 学语言：先熟悉刷题常用语言特性与标准库
- 学数据结构与算法：建立稳定的题型识别与模板能力
- 学解题：沉淀题解套路、复杂度分析与复盘方法

## 使用建议

- 按目录顺序系统学习，再结合 LeetCode 对应题单做强化
- 每个主题至少完成：核心概念、模板代码、复杂度与常见坑
- 题后复盘时记录：为什么想到这个方法、还能否优化
