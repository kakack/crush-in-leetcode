# 图（Graph）

图是由**顶点（vertex）**和**边（edge）**组成的非线性数据结构。边可以带权（权重），也可以有向或无向。LeetCode 中常见无向图、有向图、带权图，以及图的遍历、最短路径、拓扑排序等。

## 图的表示

### 邻接表（Adjacency List）

为每个顶点维护一个列表，存放与该顶点直接相连的边或邻接点。适合稀疏图，空间 O(V+E)，是 LeetCode 最常用的表示方式。

```C++
// 无向图：adj[u] 存与 u 相邻的顶点（或 (v, weight) 表示边权）
vector<vector<int>> adj(n);
// 加边 (u, v)
adj[u].push_back(v);
adj[v].push_back(u);

// 有向图：只加一条
adj[u].push_back(v);

// 带权图：邻接表存 (邻居, 边权)
vector<vector<pair<int,int>>> adj(n);
adj[u].push_back({v, w});
```

### 邻接矩阵（Adjacency Matrix）

用二维数组 `g[i][j]` 表示顶点 i 到 j 的边（或边权，无边可为 INF 或 0）。适合稠密图或需要 O(1) 查两点是否有边，空间 O(V²)。

```C++
vector<vector<int>> g(n, vector<int>(n, 0));  // 或 INF 表示无边
g[u][v] = w;  // 有向边 u->v 权为 w
```

## 图的遍历

### 深度优先搜索（DFS）

从某顶点出发，沿一条路走到底再回溯，用**栈**或**递归**实现。常用于连通块、路径存在性、环检测、拓扑排序等。

```C++
vector<bool> visited(n, false);
void dfs(int u) {
    visited[u] = true;
    for (int v : adj[u]) {
        if (!visited[v]) dfs(v);
    }
}
// 迭代版：用 stack 代替递归
```

### 广度优先搜索（BFS）

从某顶点出发，按「层」向外扩展，用**队列**实现。**无权图**上 BFS 第一次到达某点即为最短路径（边数最少）。

```C++
queue<int> q;
q.push(start);
vector<bool> visited(n, false);
visited[start] = true;
while (!q.empty()) {
    int u = q.front(); q.pop();
    for (int v : adj[u]) {
        if (!visited[v]) {
            visited[v] = true;
            q.push(v);
        }
    }
}
```

## 常见问题类型

### 连通分量

- **无向图**：对每个未访问顶点做一次 DFS/BFS，每次得到一个连通分量。也可用**并查集**维护连通性。
- **有向图**：强连通分量（SCC）常用 Tarjan 或 Kosaraju 算法。

### 拓扑排序（Topological Sort）

对有向无环图（DAG）的顶点做线性排序，使得每条有向边 (u,v) 都有 u 在 v 之前。方法：

1. **Kahn（BFS）**：统计每个点入度，将入度为 0 的点入队，每次出队并删边，新产生的入度为 0 的点入队。
2. **DFS**：对图做 DFS，在递归返回时把顶点加入结果（或逆序），即得拓扑序。

适用于依赖关系、课程先修、编译顺序等。

### 最短路径

- **无权图**：BFS 即可。
- **非负权**：Dijkstra（优先队列），每次取当前距离最小的点松弛其邻边。
- **有负权**：Bellman-Ford 或 SPFA；若存在负环则无最短路径。
- **全源**：Floyd-Warshall，O(V³)，代码简单，适合稠密图或顶点数不大时。

### 最小生成树（MST）

- **Kruskal**：按边权从小到大排序 + 并查集判环；
- **Prim**：从任一点出发逐步扩展当前生成树，常配合优先队列；
- 高频：1584、1135、1168。

### 其他

- **判环**：无向图用 DFS 看是否遇到已访问且非父节点的点；有向图用 DFS 颜色标记（白/灰/黑）或入度/Kahn 判断是否有剩余点。
- **二分图判定**：BFS/DFS 二染色，相邻点不能同色。
- **欧拉路径/回路**：度数为奇数的顶点个数为 0 或 2，且图连通（有向图有对应入度出度条件）。

## 小结

| 需求         | 常用方法           |
|--------------|--------------------|
| 遍历、连通块 | DFS / BFS          |
| 最短路径(无权) | BFS                |
| 最短路径(非负权) | Dijkstra         |
| 依赖顺序     | 拓扑排序           |
| 连通性维护   | 并查集             |
