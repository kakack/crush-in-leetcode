# 数组与集合

## 集合

**集合**是由确定元素构成的整体，集合中的元素**无顺序**、**不重复**（在编程中常通过哈希表或有序结构实现「去重」）。数学上集合不要求元素类型相同；在编程语言中，通常约定同一集合内元素类型一致。

## 列表与数组

**列表（线性表）**是数据项按线性顺序排列的有限序列，强调**顺序**和**可变的长度**。

**数组**是列表的一种实现方式：

- 用**索引**（下标）标识每项数据的位置，一般从 0 或 1 开始。
- 元素在内存中**连续存储**，每个元素占用相同大小的内存，因此可以**通过下标在 O(1) 时间内随机访问**。
- 在 C/C++ 中数组长度一般固定；在 Python/Java 等语言中，动态数组（如 `list`、`ArrayList`）会在容量不足时自动扩容，均摊仍为 O(1) 的尾部插入。

## 数组的常见操作与复杂度

| 操作           | 时间复杂度 | 说明                         |
|----------------|------------|------------------------------|
| 按索引访问     | O(1)       | 直接计算地址偏移             |
| 尾部追加       | O(1) 均摊  | 动态数组可能触发扩容         |
| 在中间插入/删除 | O(n)       | 需要移动后续元素             |
| 按值查找       | O(n)       | 无序时需遍历；有序可二分 O(log n) |

## 与链表的对比

- **数组**：连续存储，随机访问 O(1)，插入/删除（非尾部）O(n)。
- **链表**：非连续存储，按位置访问需遍历 O(n)，在已知节点处插入/删除 O(1)。

若需要频繁按索引访问或遍历，数组更合适；若频繁在中间插入、删除且不依赖下标，链表更合适。

## 多维数组

二维数组可视为「数组的数组」，在内存中通常按行连续存储（行主序）。常用于矩阵、网格类题目；访问 `a[i][j]` 为 O(1)。LeetCode 中常见题型：矩阵遍历、路径和、旋转、搜索等。

- 滑动窗口：连续区间最值、字符串窗口统计；
- 前缀和/差分：区间和与批量区间更新；
- 单调栈：下一个更大元素、直方图最大矩形；
- 二分答案：答案满足单调性时在值域上二分。
